# Wind 路线图
该文档展示 Wind 从 2023 年的开发关键路线图。

# 新特性：
- 社区建设
  - [x] 支持更多的用户中间件，如 session会话、gzip压缩等。
  - [x] 支持反向代理。 
  - 支持 swagger。
- 协议扩展
  - [x] 支持 Websocket。
  - [x] 支持 HTTP2。
  - [x] 支持 HTTP3。
- 服务治理
  - 支持更多的用户扩展。
- 性能优化
  - 改进小数据包情况下的服务器吞吐量。
- 用户体验优化
  - [x] 提供让用户更容易上手的开发体验。
  - [x] 提升代码生成工具的可用性。
